# Добро пожаловать

Это первый пост на новом статическом блоге. Всё собрано на **Next.js static
export**, **Tailwind v4** и брендовых токенах из проекта screwfast — скопированы
1:1, без подбора цветов на глаз.

## Что работает

- Посты — обычные Markdown-файлы в `content/posts/`.
- Публикация — это просто `git push` → GitHub Action → Pages.
- SEO на месте: `sitemap.xml`, `robots.txt`, RSS и чистые URL.

## Что дальше

Добавляй `.md`-файлы, настрой домен и подключи кнопки оплаты на `/buy`.
