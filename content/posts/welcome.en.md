# Welcome

This is the first post on the new static blog. Everything is built with
**Next.js static export**, **Tailwind v4**, and the brand design tokens from
the screwfast project — copied 1:1, no guessed colors.

## What works

- Posts are plain Markdown files in `content/posts/`.
- Publishing is just a `git push` → GitHub Action → Pages.
- SEO is covered: `sitemap.xml`, `robots.txt`, RSS, and clean URLs.

## Next steps

Add more `.md` files, set your domain, and wire up the payment buttons on `/buy`.
